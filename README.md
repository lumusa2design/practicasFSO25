# Prácticas FSO 2024 -2025

El siguiente repositorio se ha realizado como un trabajo en conjunto para la asignatura de  *Fundamentos de Sistemas Operativos* impartida en la *Universidad de Las Palmas de Gran Canaria*. 
Esperamos que pueda ser de utilidad para todos aquellos estudiantes que necesiten algún tipo de ayuda con la asignatura. Dispone de todos los retos que se han marcado para todos los trabajos, por lo que, puede usarse como una inspriacion.

## ⚠️Advertencia ⚠️

Este trabajo esta realizado por @lumusa2design y @jesus-tdd y, puede usarse como una guía para la asignatura, sin embargo, rogamos que se use con cautela, puesto que, las prácticas de la asignatura están pensadas para aprender y entender como funciona un sistema operativo.
